TechCoachDIY Project 0 Website!


Mission and Concepts

Our mission is to motivate EVERY students at age 11 and above to code. Through the process, students not only learn computational thinking, problem-solving as many coding initiates intend to do, but also learn to manage their computer, understand the nature of internet, what coding is about, and apply it. We design this program consisted of three sections. The first section is recommended before moving on to the later sections.

  1. Section 1: HTML/CSS
  2. Section 2: JavaScript
  3. Section 3: Python and Raspberry Pi

Section 1

HTML/CSS

HTML is the standard markup language for web pages and easy to learn. There is no mathematical or logical commands invovlved as most computer programmings have.

CSS works together with HTML to styles the layout of the web page by adding to HTML elements in three ways: inline, internal, and external.

TechCoachDIY recommends to write HTML/CSS as the first step of learning text coding. This is based on successful student outcomes (page 38-39) from approximately 400 7th and 8th grade students. Students were engaged, feeling rewarded, and producing good quality of works. Students will be guided to learn through free on-line resource (e.g. codecademy) and work on designed projects.

Section 2

JavaScript

JavaScript enables interactive web pages from the client-side page. The main goal here is to learn mathematical function and logical commands through JavaScript while applying JavaScript to create dynamic web page. Students will be guided to learn through free on-line resource (e.g. codeHS) and work on designed projects.

Section 3

Python and Raspberry Pi

The Raspberry Pi is a series of small single-board computers developed by the Raspberry Pi Foundation in UK. Python is its default progarmming language. The purpose of this section is to learn how computer works and hardware/software communication.
